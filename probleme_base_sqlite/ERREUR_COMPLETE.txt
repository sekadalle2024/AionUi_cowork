CONTEXTE :
----------
- Projet : AionUi v1.8.25
- Electron : 37.3.1 (utilise Node.js v22.x)
- Module problématique : better-sqlite3 (module natif C++)
- NODE_MODULE_VERSION 127 = Node.js v20.x (version compilée)
- NODE_MODULE_VERSION 136 = Node.js v22.x (version requise par Electron)

SERVICES AFFECTÉS :
-------------------
[Database] Initializing database - ÉCHEC

IMPACT :
--------
- Impossible de créer ou charger des conversations
- Base de données inaccessible
- Aucune donnée utilisateur ne peut être sauvegardée
- L'application démarre mais est inutilisable

FRÉQUENCE :
-----------
L'erreur se répète à chaque tentative d'accès à la base de données, générant
des centaines de lignes d'erreur dans les logs.

TENTATIVES ÉCHOUÉES :
---------------------
1. npm rebuild better-sqlite3 --build-from-source
   → Recompile pour Node.js standard, pas Electron

2. npm install (réinstallation complète)
   → Télécharge les binaires précompilés pour Node.js standard

3. Suppression de node_modules et réinstallation
   → Même problème, binaires pour mauvaise version

4. npx electron-rebuild -f -w better-sqlite3
   → Échoue sur node-pty avec erreur Visual Studio

SOLUTION QUI FONCTIONNE :
-------------------------
npx electron-builder install-app-deps

Cette commande :
1. Détecte automatiquement la version d'Electron (37.3.1)
2. Identifie la version de Node.js correspondante (v22.x)
3. Recompile better-sqlite3 pour NODE_MODULE_VERSION 136
4. Recompile également keytar (autre module natif)

COMMANDES COMPLÈTES :
---------------------
# Windows PowerShell
Remove-Item -Recurse -Force out -ErrorAction SilentlyContinue
Remove-Item -Recurse -Force node_modules\better-sqlite3\build -ErrorAction SilentlyContinue
npx electron-builder install-app-deps
.\start-dev.ps1

# macOS/Linux Bash
rm -rf out
rm -rf node_modules/better-sqlite3/build
npx electron-builder install-app-deps
npm start

PRÉVENTION :
------------
Exécuter après chaque :
- npm install (réinstallation des dépendances)
- Mise à jour d'Electron
- Mise à jour de better-sqlite3
- Clone du projet sur nouvelle machine
